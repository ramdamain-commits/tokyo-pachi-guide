# 取材予定の確認手順

「今週どの店に取材が入るか」を調べるときの巡回手順。取材予定は日々変わるため、このリポジトリではデータとして持たず、確認手順とリンク集だけを整備する（正データは `data/media.json` の `scheduleUrl`）。

## 手順

1. **集約サイトで当たりを付ける**
   - [P-WORLD 取材・来店情報](https://www.p-world.co.jp/hall/interviews) — 複数媒体を横断掲載
   - [でちゃう!PLUS スケジュール](https://www.dechau.com/schedule/schedule_all.php) — 全国の取材・来店を集約
2. **媒体公式スケジュールで裏取りする**（下記リンク集。集約サイトの転記ミス・漏れがあるため公式を正とする）
3. **特定日と重なる店を確認する** — ビューワの「日付指定」タブでその日の該当店を出し、取材予定と重なる店を優先候補にする

## 読み方の注意

- 公約・傾向は伝聞であり「〜とされる」以上の確度はない（風営法上、設定示唆の公式イベントは存在しない）
- 掲載が無い＝取材が無い、ではない（欠測をデフォルト推定しない）
- 過去実績ベースの「入りやすい媒体」は `data/coverage.json`（ビューワの取材傾向タブ）を参照。予定と実績は別物として扱う
