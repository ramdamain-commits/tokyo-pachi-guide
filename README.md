# 東京パチ・スロ特定日ガイド

東京都内の主要エリアのパチンコ・パチスロ店について、旧イベント日（特定日）パターンと店舗概要をローカルで即答できるようにする個人用ガイド。「今日どの店が該当日か」を毎回AIに聞かずに済ませるのが目的。

**公開方針**: 2026-07-17 に非公開→公開へ変更（本人判断・当初の「非公開」記述を上書き）。GitHub 公開リポジトリ＋GitHub Pages でビューワを公開し、portal からも参照する。掲載データは店舗の公開情報・第三者サイトの過去傾向推定のみで、個人情報（居住エリア・実名等）は含めない。旧イベント日は推定であり店舗の公式情報ではない点に留意（下記「注意」参照）。

## 構成

- `data/stores.json` — 店舗マスタ（エリア・台数・特定日ルール・抽選方式・閉店時刻・換金率・概要・出典・確認日）。2026-07-16 に全32店で生成・詳細補完まで完了（上野4/秋葉原5/新宿6/渋谷4/池袋8/新橋3/京橋銀座2）
- `data/media.json` — 取材媒体マスタ（媒体名・取材傾向・対象ジャンル・確度・出典）。店舗単位ではなく媒体単位の一般的傾向（伝聞情報、公式公約ではない）
- `match.mjs` — 日付マッチングロジック本体（`rules[].type` 全7種の判定・`ruleLabel` によるバッジ用日本語ラベル生成）
- `index.html` / `app.js` — ビューワ（今日/明日/日付指定/エリア別一覧/取材傾向の5タブ）。`app.js` は `<script type="module">` として `match.mjs` を import する
- `test/match.test.mjs` — `match.mjs` のユニットテスト（`node --test` で実行、境界ケース網羅・全18件pass）
- `docs/superpowers/specs/` — 設計書

## 閲覧方法

公開ビューワ: **https://ramdamain-commits.github.io/tokyo-pachi-guide/**

ローカルで動かす場合、ES module + fetch を使うため `file://` では動作しない。リポジトリルートで以下を実行してブラウザから `http://localhost:8099/` を開く。

```
python3 -m http.server 8099
```

## 対象エリア

上野・秋葉原・新宿・新橋・池袋・渋谷・京橋/銀座近辺（7エリア、30〜40店規模）

## 注意

旧イベント日は第三者サイトの過去傾向推定であり、店舗の公式情報ではない（風営法上、設定示唆の公式イベントは存在しない）。データには必ず出典URLと確認日を付け、四半期に1回棚卸しする。

## 更新方法

特定日はパターンルール（「5のつく日」等）のため原則更新不要。店舗の閉店・リニューアル・傾向変化があれば `data/stores.json` を修正する。
